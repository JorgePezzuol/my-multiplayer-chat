# Project
Multiplayer game with Phaser 3, embedded Jitsi SDK and real time chat with Socket io

## Technologies
<ul>
  <li>Phaser 3</li>
  <li>Node</li>
  <li>Socket io</li>
  <li>WebRTC</li>
</ul>

## Setup
* clone/download and cd my-multiplayer-chat
* node server.js
